HTTP和WebSocket的代理设置

2020-06-13
  • 代理

背景

最近在项目工程实施的过程中，需要使项目可以获取一些信息，于是想到如何通过代理获取。

问题点

  1. 如何能够使所有的web请求的时候都走代理呢？

解决过程

  1. 能否让自己的所有软件都走代理？
    -可行，只要软件可以设置代理。
  2. 如果1可行，是否可以让Intellij走代理？
    -可行，但是无法让Web工程全部走代理。
  3. 既然2不可行，能否让JVM走代理。
    -不知道，没找到方案。
  4. 那是否可以让HTTP请求走代理？
    -可行，在创建RestTemplate对象的时候设置Proxy
  5. WebSocket是否可行？
    -可行，在创建WebSocket的时候设置Proxy即可。

代码

根据4,5点，可以在创建RestTemplate对象和WebSocketClient对象时候，设置代理即可。
